Application web de gestion de rapports de stages

Le but de ce projet est de travailler les connexions en PHP avec la BDD, les différentes interfaces, les différents affichages en fonction de plusieurs facteurs.
Pour se faire, différents utilisateurs ont été implémentés et chacun possède des fonctionnalités qui lui sont propres.

L'application permet aux éléves d'entrer chaque jour le rapport de la journée, ce qu'ils ont fais durant la journée. Ils sont capables de le modifier et de modifier la date de leur poste.
